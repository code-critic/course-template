# Problems

In this folder are stored problems - their descriptions, inputs, outputs and 
reference solution (if there is any)


A structure is as follows:
```txt
2019/problems
├── problem-1
│   ├── intput
│   │   ├── case-1
│   │   ├── case-2
│   │   └── case-n
│   └── output
│       ├── case-1
│       ├── case-2
│       └── case-n
├── problem-2
│   ├── intput
│   │   ├── case-1
│   │   ├── case-2
│   │   └── case-n
│   └── output
│       ├── case-1
│       ├── case-2
│       └── case-n
└── problem-n
    ├── intput
    │   ├── case-1
    │   ├── case-2
    │   └── case-n
    └── output
        ├── case-1
        ├── case-2
        └── case-n
```
